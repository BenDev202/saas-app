# SaaS App

A simple SaaS application.

## Features

- User authentication
- Dashboard
- Subscription management

## Getting Started

1. Clone the repo:
  ```
  git clone https://github.com/BenDev202/saas-app.git
  ```
2. Install dependencies:
  ```
  npm install
  ```
3. Start the app:
  ```
  npm start
  ```
